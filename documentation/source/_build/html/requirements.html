Requirements¶

Important

simple_tide_report requires Python 3.x and was developed under Python 3.6. As such they are unlikely to work with Python 2.x.

Special Libraries¶

simple_tide_report makes use of the following ‘homegrown’ modules:

  • ibtidelib

A version of this has been included as part of the package.

Additional Modules¶

In addition to the standard Python 3.6 Modules simple_tide_report makes use of the following packages:

  • requests

  • tqdm
